# GPU Usage

GPU minutes consumed by your trainings against your plan's limit.

---

## Overview

The GPU Usage card on the [Dashboard](/docs/dashboard) shows GPU minutes used vs. your plan's allowance. GPU time is consumed whenever a training runs on GPU-backed compute.

---

## Reducing GPU Usage

- Reserve GPU training for jobs that actually need it — use CPU for quick iterations
- Avoid re-running trainings with unchanged settings/data
- See [Plans](/docs/dashboard/plans) to upgrade for more GPU minutes

---

## Troubleshooting

- **My GPU usage jumped unexpectedly.**
  Check [Training](/docs/training) for any runs left unintentionally active or repeated.

- **My GPU minutes didn't reset after my billing cycle renewed.**
  Refresh the Dashboard. If it still looks wrong, contact support.
