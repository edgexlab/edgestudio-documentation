# CPU Usage

CPU minutes consumed by your trainings against your plan's limit.

---

## Overview

The CPU Usage card on the [Dashboard](/docs/dashboard) shows CPU minutes used vs. your plan's allowance. CPU time is consumed whenever a training runs on CPU rather than GPU.

---

## Reducing CPU Usage

- Use smaller datasets or fewer epochs while iterating
- Favor GPU-backed training for larger jobs where available
- See [Plans](/docs/dashboard/plans) to upgrade for more CPU minutes

---

## Troubleshooting

- **My CPU usage jumped unexpectedly.**
  Check [Training](/docs/training) for any runs left unintentionally active or repeated.

- **My CPU minutes didn't reset after my billing cycle renewed.**
  Refresh the Dashboard. If it still looks wrong, contact support.
